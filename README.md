## 프로젝트 개요
- 배경색, 아이콘, 글자색 등으로 날씨를 표현할 수 있는 날씨웹, 앱
- Openweather API 를 사용해 날씨 정보를 가져와 현재 날씨 정보에 맞게 아이콘 및 날씨 관련단에 자동 변경
- Anchor 태그를 이용해 페이지 전환
